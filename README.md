# Remotion Test Project

This is a Remotion project for creating animations of a macOS terminal window.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the Remotion Studio:
```bash
npm start
```

## Composition: MacTerminal

- **Dimensions**: 1280x1000px
- **Duration**: 180 frames (6 seconds at 30fps)
- **Theme**: Light theme macOS Terminal window

The composition displays a clean macOS-style terminal window with:
- macOS traffic light buttons (red, yellow, green)
- Window title bar
- Light-themed terminal background
- Empty prompt with blinking cursor
- Git branch indicator
